Project Granth

Github | Heroku


A web app where users can annotate and discuss Sikh poetry. I wanted to create a space where the user can better understand metaphors and analogies in order to spark conversation with others about how poetry connects to their everyday life. I built both the backend webserver/database and the frontend user interface.

Technologies: JavaScript, jQuery, EJS, Express, Node.js, Mongo DB, Heroku
